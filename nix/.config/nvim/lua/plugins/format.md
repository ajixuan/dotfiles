# Required Formatters

| Language   | Formatter     | Install                                         |
|------------|---------------|--------------------------------------------------|
| Lua        | stylua        | `cargo install stylua` or download from releases |
| Python     | ruff          | `pip install ruff` or download from releases     |
| Go         | gofmt         | Comes with Go (`apt install golang-go`)          |
| Rust       | rustfmt       | Comes with Rust (`rustup component add rustfmt`) |
| Bash       | shfmt         | Download from GitHub releases                    |
| JS/TS/JSON | prettier      | `npm install -g prettier`                        |
| JS/TS/JSON | dprint        | `cargo install dprint` (alternative without npm) |
| YAML       | prettier      | `npm install -g prettier`                        |
| YAML       | dprint        | `cargo install dprint` (alternative without npm) |
| Markdown   | prettier      | `npm install -g prettier`                        |
| Markdown   | dprint        | `cargo install dprint` (alternative without npm) |
